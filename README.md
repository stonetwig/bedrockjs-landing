# BedrockJS Landing Page

A simple landing page built with BedrockJS and served with Deno.

## Getting started

```bash
deno task bundle
deno task dev
```

Then open `http://localhost:8000`.

## Project structure

- `src/client.ts` - BedrockJS app shell and router setup.
- `src/pages/` - Page components for routes.
- `src/components/` - Reusable components.
- `public/` - Static assets and styles.
